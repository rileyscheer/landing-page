What is this website about?

Space! I built this website to improve on my HTML and CSS skills. Space just happened to be the topic I decided on. This website originally started as just a landing page but I have expanded on it as I have learned new things. This website contains non functional buttons on the main page because I wanted to practice styling them and adding effects with ":hover" in CSS. They are non functional at the moment because I am just now beginning to learn JavaScript and do not know how to store information yet. Once I learn that, I may come back to update the site to have accounts and other features but I will be building bigger projects by then so time will tell. Also, this site is made to be viewed on desktop (2560x1440p) but I hope to come back and update it for mobile soon.
